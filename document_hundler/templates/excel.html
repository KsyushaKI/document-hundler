Выберете несколько файлов формата .xls,.xlsx и затем нажмите на кнопку "Получить Excel".

* Во всех файлах один из столбцов должен быть под названием "Номер". Именно по нему будет производиться сортировка;

* Все необходимые файлы должны находиться в одной и той же папке;

* Зажмите Ctrl(Windows) / cmd(Mac) для выбора, а затем кликайте на каждый необходимый файл, чтобы его отметить для выбора, после завершения нажмите на кнопку "Открыть".
